Shigoto UI

日本の業務系 SaaS のための Web Components 主体の UI ライブラリ。 帳票・申請・承認・マスタ管理など、定型業務 UI を最短距離で組み立てるための部品が揃っています。

導入

npm

npm install @shigoto-ui/elements

使用したい部品だけ個別にインポートします（ツリーシェイク対応）。

import '@shigoto-ui/elements/button';
import '@shigoto-ui/elements/money-input';

CDN

<link rel="stylesheet" href="https://unpkg.com/@shigoto-ui/elements/dist/shigoto-ui.css" />
<script type="module" src="https://unpkg.com/@shigoto-ui/elements"></script>

基本セットアップ

ルート要素に data-sgt-theme="light" を設定し、 CSS 変数で配色・余白・フォントを上書きできます。 

:root {
  --accent: #2b63e0;     /* 主要色 */
  --ink: #0f172a;        /* 文字色 */
  --paper: #ffffff;      /* 背景色 */
  --line: #e2e8f0;       /* 罫線 */
  --r-sm: 3px;           /* 角丸（小） */
}

フレームワーク連携

  • HTML / 素のJS: @shigoto-ui/elements を読み込めばそのまま使用可
  • React: @shigoto-ui/react がラッパを提供
  • Vue: @shigoto-ui/vue でテンプレ統合

各コンポーネントの使用例はコンポーネントカタログを参照してください。
